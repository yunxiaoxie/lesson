二、数组如何画不规则图形(点阵映射)
    2.1 0  不映射
    2.2 1  映射
三、确定数组的维度
    3.1 要绘制数学0-9
    3.2 要绘制内容长度11
    3.3 要绘制的行数以及列表  10*7
    综不所述，确定三维数组包含的是0-9以及：的二维点阵
四、借助canvas绘制
    4.1 确定圆开的画法 cxt.arc(x,y,r,0,2PI,false)
    4.2 确定圆的属性(矩形辅助)
    设圆的半径为:R, 圆与矩形间隔为1
    则一个矩形的宽高：2(R+1)
    整个数字的宽度为2(R+1)*7 高度为2(R+1)*10
    若数字的高度为100,则圆形的半径为：100=20(R+1)
    4.3把数组里的1转换成圆形(求xy)
        4.3.1确定圆形的x轴坐标
        R+1
        2*(R+1)+(R+1)
        4*(R+1)+(R+1)
        6*(R+1)+(R+1)
        ...
        综上所述，得到j*2*(R+1)+(R+1)
        4.3.2确定圆形的y轴坐标
        R+1
        2*(R+1)+(R+1)
        4*(R+1)+(R+1)
        6*(R+1)+(R+1)
        ...
        综上所述，得到i*2*(R+1)+(R+1)
        4.3.3将绘制的数组在x轴进行平辅
            4.3.3.1自定义字间隔100
            4.3.3.2最好与R挂钩,用index进行位置规律的偏移14*(this.R+2)
